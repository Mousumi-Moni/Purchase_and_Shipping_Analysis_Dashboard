Project Summary

This project involved building an end-to-end Purchase and Shipping Analysis Dashboard in Microsoft Excel. Starting from raw transactional data, I designed a fully interactive dashboard with pivot tables, dynamic charts, and slicers to support data-driven decision-making for a retail business.
The dashboard consolidates $578,951 in total purchases across 10,745 customer transactions, spanning four product categories, multiple payment methods, and two fiscal years (2023–2024).
